Controller mappings

Default controller layout (A/Xbox):

  • A: Shoot / Confirm / Respawn
  • B: Ship special (contextual)
  • X: Teleport
  • Y: Shield toggle
  • LB: Rift, RB: Phase
  • LT+RT: Show this mapping
  • Start: Start / Enter, Back: Chunk Select
  • Left stick / D-pad: Move / title navigation
